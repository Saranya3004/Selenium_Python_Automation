# NopCommerce Automation Testing

## About
Automation test framework for NopCommerce e-commerce application.

## Tech Stack
- Python
- Selenium WebDriver
- PyTest
- BDD/Cucumber (Behave)
- Page Object Model (POM)
- Postman (API Testing)

## Test Coverage
- User Registration & Login
- Product Search & Add to Cart
- Checkout & Payment Workflow
- End-to-End Regression Testing
